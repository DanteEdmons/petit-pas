# План: ультимативный курс сербского языка

## Фаза 1 — Обогащение словаря (контекст и примеры)

### Задача 1.1: Добавить примеры использования ко всем 399 словам
Каждое слово получает поле `example` — предложение на сербском с переводом.
```json
{
  "front": "човек / čovek",
  "back": "человек",
  "example": "Тај човек је мој комшија. — Тот человек мой сосед."
}
```

### Задача 1.2: Добавить латиницу ко всем словам
Сербский использует ОБЕ письменности (кириллица и латиница). Это критично.
Формат `front` становится: `"кириллица / latinica"` для всех слов на всех уровнях.

### Задача 1.3: Добавить грамматические метки
- Существительные: род (м/ж/ср) в поле `frontHint`
- Глаголы: вид (свр./несвр.) в поле `frontHint`
- Прилагательные: формы м/ж/ср

Пример:
```json
{
  "front": "човек / čovek",
  "back": "человек",
  "frontHint": "м.р.",
  "example": "Тај човек је мој комшија. — Тот человек мой сосед."
}
```

## Фаза 5 — Тренажёр падежей (новая фича, специфична для сербского)

### Задача 5.1: Создать данные для тренажёра падежей
В serbian.json добавить новую секцию `cases` на каждом уровне:
```json
"cases": [
  {
    "noun": "жена",
    "gender": "ж",
    "forms": {
      "nominativ": "жена",
      "genitiv": "жене",
      "dativ": "жени",
      "akuzativ": "жену",
      "vokativ": "жено",
      "instrumental": "женом",
      "lokativ": "жени"
    }
  }
]
```
Beginner: 20 существительных (м/ж/ср род, ед. число)
Intermediate: 20 существительных (ед. + мн. число)
Advanced: 15 существительных (исключения и неправильные формы)

### Задача 5.2: Создать UI тренажёра падежей в main.js
Режимы:
- **Просклоняй**: показывается слово + падеж → пользователь пишет форму
- **Определи падеж**: показывается предложение с выделенным словом → выбери падеж
- **Заполни пропуск**: предложение с пропуском → вставь слово в нужном падеже

## Фаза 7 — Улучшение SRS (уровни уверенности)

### Задача 7.1: Заменить 2 кнопки на 4 в SRS
Вместо «Знаю / Не знаю» → «Не помню / Трудно / Нормально / Легко».
Каждая кнопка даёт разный множитель интервала:
- Не помню: box = 0 (как сейчас)
- Трудно: box не меняется, интервал × 0.5
- Нормально: box + 1 (как сейчас)
- Легко: box + 2, интервал × 1.5

### Задача 7.2: Обновить store.js для difficulty-aware SRS
