Перфект (Perfekt)

Перфект — основное прошедшее время в сербском языке. Образуется с помощью вспомогательного глагола бити (biti) в настоящем времени и активного причастия на -л/-ла/-ло.

бити (презент) + причастие на -л

Перфект состоит из двух компонентов: вспомогательного глагола бити в настоящем времени и активного причастия прошедшего времени.

Краткие формы глагола бити / biti

ја (я) сам / sam
ми (мы) смо / smo
ти (ты) си / si
ви (вы) сте / ste
он/она/оно је / je
они/оне/она су / su

Полные формы глагола бити / biti

ја јесам / jesam
ми јесмо / jesmo
ти јеси / jesi
ви јесте / jeste
он/она/оно јесте / jeste
они/оне/она јесу / jesu
Важно: В утвердительных предложениях обычно используются краткие формы (сам, си, је...). Полные формы (јесам, јеси, јесте...) употребляются для эмфазы, в начале предложения или в вопросах.

Активное причастие образуется от основы инфинитива. Окончание зависит от рода и числа:

Окончания причастия

м.р. ед.ч. -о / -ао
мн.ч. (м.р.) -ли
ж.р. ед.ч. -ла
мн.ч. (ж.р.) -ле
ср.р. ед.ч. -ло
мн.ч. (ср.р.) -ла

Примеры образования причастий

радити / raditi (работать) радио / radio
певати / pevati (петь) певао / pevao
ићи / ići (идти) ишао / išao
доћи / doći (прийти) дошао / došao
моћи / moći (мочь) могао / mogao
рећи / reći (сказать) рекао / rekao
Особенности: У глаголов на -ти с основой на согласный (ићи, доћи) причастие образуется от особой основы. Глаголы на -овати/-евати теряют -ова-/-ева- перед -л: радовати → радовао.

радити / raditi (работать)

ја (я) сам радио / радила / sam radio / radila
ми (мы) смо радили / smo radili
ти (ты) си радио / радила / si radio / radila
ви (вы) сте радили / ste radili
он/она/оно је радио / радила / радило / je radio / radila / radilo
они/оне/она су радили / su radili

ићи / ići (идти)

ја (я) сам ишао / ишла / sam išao / išla
ми (мы) смо ишли / smo išli
ти (ты) си ишао / ишла / si išao / išla
ви (вы) сте ишли / ste išli
он/она/оно је ишао / ишла / ишло / je išao / išla / išlo
они/оне/она су ишли / su išli

доћи / doći (прийти)

ја (я) сам дошао / дошла / sam došao / došla
ми (мы) смо дошли / smo došli
ти (ты) си дошао / дошла / si došao / došla
ви (вы) сте дошли / ste došli
он/она/оно је дошао / дошла / дошло / je došao / došla / došlo
они/оне/она су дошли / su došli
Ја сам радио цео дан. / Ja sam radio ceo dan.
Я работал весь день. (м.р.)
Она је учила српски. / Ona je učila srpski.
Она учила сербский.
Ми смо живели у Београду. / Mi smo živeli u Beogradu.
Мы жили в Белграде.

Причастие обязательно согласуется с подлежащим в роде и числе:

м.р. ед.ч. -о / -ао
м.р. мн.ч. -ли
ж.р. ед.ч. -ла
ж.р. мн.ч. -ле
ср.р. ед.ч. -ло
ср.р. мн.ч. -ла
Марко је дошао. / Marko je došao.
Марко пришёл. (м.р.)
Ана је дошла. / Ana je došla.
Ана пришла. (ж.р.)
Дете је дошло. / Dete je došlo.
Ребёнок пришло. (ср.р.)
Они су дошли. / Oni su došli.
Они пришли. (м.р. мн.ч.)
Оне су дошле. / One su došle.
Они пришли. (ж.р. мн.ч.)
Деца су дошла. / Deca su došla.
Дети пришли. (ср.р. мн.ч.)
Важно: Во множественном числе для смешанной группы (мужчины и женщины) используется мужская форма на -ли.

Отрицание образуется полными формами глагола бити с приставкой ни-:

Отрицательные формы глагола бити

ја (я) нисам / nisam
ми (мы) нисмо / nismo
ти (ты) ниси / nisi
ви (вы) нисте / niste
он/она/оно није / nije
они/оне/она нису / nisu

Спряжение «радити» в отрицании

ја нисам радио/радила / nisam radio/radila
ми нисмо радили / nismo radili
ти ниси радио/радила / nisi radio/radila
ви нисте радили / niste radili
он/она није радио/радила / nije radio/radila
они нису радили / nisu radili
Ја нисам спавао добро. / Ja nisam spavao dobro.
Я не спал хорошо.
Она није дошла на време. / Ona nije došla na vreme.
Она не пришла вовремя.
Они нису разумели. / Oni nisu razumeli.
Они не поняли.

В вопросах используются полные формы вспомогательного глагола или частица ли:

Јеси ли радио данас? / Jesi li radio danas?
Ты работал сегодня?
Је ли она дошла? / Je li ona došla?
Она пришла?
Шта си урадио? / Šta si uradio?
Что ты сделал?
Где си био? / Gde si bio?
Где ты был?
Када су они стигли? / Kada su oni stigli?
Когда они прибыли?
Частица «ли»: В общих вопросах (да/нет) частица «ли» ставится после вспомогательного глагола: Јеси ли?, Је ли?, Јесу ли? В разговорной речи может опускаться.

Краткие формы вспомогательного глагола (сам, си, је...) являются энклитиками и занимают вторую позицию в предложении:

Ја сам радио. / Ja sam radio.
Я работал. (местоимение + сам)
Данас сам радио. / Danas sam radio.
Сегодня я работал. (наречие + сам)
У школу сам ишао. / U školu sam išao.
В школу я ходил. (предлог + сущ. + сам)

Важные правила

  • Краткие формы (сам, си, је...) стоят на второй позиции после первого значимого слова
  • Полные формы (јесам, јеси...) могут стоять в начале предложения (вопросы, эмфаза)
  • При отрицании используются только полные формы (нисам, ниси...)
  • Причастие обычно стоит в конце предложения
Энклитики: Краткие формы бити не могут стоять в начале предложения (кроме вопросов с «ли»). Если подлежащего нет, на первое место ставится другое слово или используется полная форма.

Перфект — наиболее употребительное прошедшее время в современном сербском языке:

  • Завершённые действия в прошлом: Ја сам прочитао књигу. / Ja sam pročitao knjigu.
  • Действия с результатом в настоящем: Он је дошао. / On je došao. (он сейчас здесь)
  • Прошлый опыт: Она је била у Паризу. / Ona je bila u Parizu.
  • Серия завершённых действий: Устао сам, доручковао и отишао на посао. / Ustao sam, doručkovao i otišao na posao.
Учио сам српски две године. / Učio sam srpski dve godine.
Я учил сербский два года.
Видели смо тај филм. / Videli smo taj film.
Мы видели этот фильм.
Шта си јео за доручак? / Šta si jeo za doručak?
Что ты ел на завтрак?
Сравнение с русским: Сербский перфект соответствует русскому прошедшему времени совершенного и несовершенного вида. В разговорной речи перфект практически полностью вытеснил аорист и имперфект.
