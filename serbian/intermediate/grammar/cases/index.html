Падежи в сербском языке (Падежи / Padeži)

В сербском языке 7 падежей, аналогично русскому, но с некоторыми отличиями. Изучение падежных окончаний - ключ к правильному построению предложений.

Вопросы: ко? / што? (ko? / što?) - кто? / что?

Именительный падеж - это основная форма существительного. Используется для подлежащего.

Окончания существительных в номинативе:

Род Единственное число Множественное число
Мужской согласный, -о, -е -и, -ови, -еви
Женский -а, согласный -е, -и
Средний -о, -е -а
Човек ради. / Čovek radi.
Человек работает.
Жена пева. / Žena peva.
Женщина поет.
Дете игра. / Dete igra.
Ребенок играет.

Вопросы: кога? / чега? (koga? / čega?) - кого? / чего?

Используется для выражения принадлежности, отрицания, после предлогов (од, из, без, до, код, поред и др.).

Окончания в генитиве:

Род Единственное число Множественное число
Мужской -а, -у -а
Женский -е, -и нулевое окончание
Средний -а нулевое окончание

Примеры склонения:

Nominativ Genitiv ед.ч. Genitiv мн.ч.
град / grad (город) града / grada градова / gradova
жена / žena (женщина) жене / žene жена / žena
село / selo (деревня) села / sela села / sela
Нема воде. / Nema vode.
Нет воды.
Књига мог брата. / Knjiga mog brata.
Книга моего брата.
Из Београда сам. / Iz Beograda sam.
Я из Белграда.
Важно: В сербском языке генитив используется при отрицании глаголов (как в русском): Немам времена (нет времени), а не *немам време.

Вопросы: коме? / чему? (kome? / čemu?) - кому? / чему?

Используется для непрямого дополнения, после предлогов к, према, благодаря.

Окончания в дативе:

Род Единственное число Множественное число
Мужской -у -има
Женский -и -ама
Средний -у -има

Примеры склонения:

Nominativ Dativ ед.ч. Dativ мн.ч.
пријатељ / prijatelj (друг) пријатељу / prijatelju пријатељима / prijateljima
сестра / sestra (сестра) сестри / sestri сестрама / sestrama
дете / dete (ребенок) детету / detetu деци / deci
Дајем књигу брату. / Dajem knjigu bratu.
Даю книгу брату.
Иде ми добро. / Ide mi dobro.
У меня идет хорошо.
Према мени... / Prema meni...
По отношению ко мне...

Вопросы: кога? / што? (koga? / što?) - кого? / что?

Используется для прямого дополнения, после предлогов движения (у, на, за, кроз, низ, уз).

Окончания в акузативе:

Род Единственное число Множественное число
Мужской (одуш.) -а -е
Мужской (неодуш.) = Nominativ -е
Женский (-а) -у -е
Женский (согл.) = Nominativ -и
Средний = Nominativ -а

Примеры склонения:

Nominativ Akuzativ ед.ч. Akuzativ мн.ч.
брат / brat (брат) брата / brata браћу / braću
стол / sto (стол) сто / sto столове / stolove
кућа / kuća (дом) кућу / kuću куће / kuće
Видим брата. / Vidim brata.
Вижу брата.
Читам књигу. / Čitam knjigu.
Читаю книгу.
Идем у школу. / Idem u školu.
Иду в школу (направление).
Важно: Различие одушевленности/неодушевленности важно только для мужского рода!

Используется для обращения

В отличие от русского языка, где звательный падеж практически исчез, в сербском он активно используется.

Окончания в вокативе:

Род Единственное число Множественное число
Мужской -е, -у, -о -и
Женский (-а) -о, -е -е
Средний -е, -о -а

Примеры склонения:

Nominativ Vokativ ед.ч. Vokativ мн.ч.
Милан / Milan Милане / Milane -
Марко / Marko Марко / Marko -
Ана / Ana Ано / Ano -
господин / gospodin господине / gospodine господо / gospodo
Марко, дођи овамо! / Marko, dođi ovamo!
Марко, иди сюда!
Добар дан, господине! / Dobar dan, gospodine!
Добрый день, господин!
Мајко, где си? / Majko, gde si?
Мама, где ты?
Правило: Имена на -о, -е обычно не меняются в вокативе. Имена на согласный получают окончание -е.

Вопросы: ким? / чим? (kim? / čim?) - кем? / чем?

Используется для обозначения инструмента, средства действия, после предлогов (с, са, између, над, под, пред, за).

Окончания в инструментале:

Род Единственное число Множественное число
Мужской -ом, -ем -има
Женский -ом, -ом -ама
Средний -ом, -ем -има

Примеры склонения:

Nominativ Instrumental ед.ч. Instrumental мн.ч.
нож / nož (нож) ножем / nožem ножевима / noževima
рука / ruka (рука) руком / rukom рукама / rukama
перо / pero (перо) пером / perom перима / perima
Пишем оловком. / Pišem olovkom.
Пишу карандашом.
Идем са братом. / Idem sa bratom.
Иду с братом.
Он је учитељ. / On je učitelj.
Он учитель. (профессия в инструментале)
Важно: При обозначении профессии в сербском часто используется номинатив, но инструментал также возможен, особенно после глагола "постати" (стать).

Вопросы: где? / о коме? / о чему? (gde? / o kome? / o čemu?)

Используется для обозначения места (где?), после предлогов у, на, о, по, при.

Окончания в локативе:

Род Единственное число Множественное число
Мужской -у -има
Женский -и -ама
Средний -у -има

Примеры склонения:

Nominativ Lokativ ед.ч. Lokativ мн.ч.
град / grad (город) граду / gradu градовима / gradovima
школа / škola (школа) школи / školi школама / školama
село / selo (деревня) селу / selu селима / selima
Живим у Београду. / Živim u Beogradu.
Живу в Белграде.
Причамо о филму. / Pričamo o filmu.
Говорим о фильме.
Књига је на столу. / Knjiga je na stolu.
Книга на столе.
Важно: Предлоги У и НА с локативом означают местонахождение (где?), а с акузативом - направление (куда?). Сравните: У ШКОЛИ (в школе - где?) vs. У ШКОЛУ (в школу - куда?).

Сходство с русским: Система падежей очень похожа на русскую, но есть отличия в окончаниях и употреблении некоторых предлогов. Звательный падеж активно используется в сербском, тогда как в русском почти исчез.
