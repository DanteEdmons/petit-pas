Падежи (Padeži)

В сербском 7 падежей — как в русском, но с другими окончаниями. Раскройте тему, чтобы увидеть окончания, склонение и примеры.

Вопросы: ко? / што? (кто? / что?)

Основная, словарная форма. Используется для подлежащего.

Окончания

м.р.согласный, -о, -е ж.р.-а, согласный ср.р.-о, -е
м.р.-и, -ови, -еви ж.р.-е, -и ср.р.-а
Човек ради. / Čovek radi.
Человек работает.
Жена пева. / Žena peva.
Женщина поет.
Дете игра. / Dete igra.
Ребенок играет.

Вопросы: кога? / чега? (кого? / чего?)

Принадлежность, отрицание, после предлогов: од, из, без, до, код, поред.

Окончания

м.р. ед.-а ж.р. ед.-е, -и ср.р. ед.-а м.р. мн.-а, -ова ж.р. мн.-а / ∅ ср.р. мн.-а / ∅

Примеры склонения

град → генитивграда / grada
мн.ч.градова / gradova
жена → генитивжене / žene
мн.ч.жена / žena
село → генитивсела / sela
мн.ч.села / sela
Нема воде. / Nema vode.
Нет воды.
Књига мог брата. / Knjiga mog brata.
Книга моего брата.
Из Београда сам. / Iz Beograda sam.
Я из Белграда.
Как в русском: Генитив используется при отрицании — Немам времена (нет времени).

Вопросы: коме? / чему? (кому? / чему?)

Непрямое дополнение. Предлоги: ка, према.

Окончания

м.р. ед.-у ж.р. ед.-и ср.р. ед.-у м./ср. мн.-има ж.р. мн.-ама

Примеры склонения

пријатељ → дативпријатељу / prijatelju
мн.ч.пријатељима / prijateljima
сестра → дативсестри / sestri
мн.ч.сестрама / sestrama
дете → дативдетету / detetu
мн.ч.деци / deci
Дајем књигу брату. / Dajem knjigu bratu.
Даю книгу брату.
Иде ми добро. / Ide mi dobro.
У меня хорошо.
Према мени... / Prema meni...
По отношению ко мне...

Вопросы: кога? / шта? (кого? / что?)

Прямое дополнение. Предлоги движения: у, на, за, кроз, низ, уз.

Окончания

м.р. одуш.-а м.р. неодуш.= Ном. ж.р. (-а)-у ж.р. (согл.)= Ном. ср.р.= Ном.

Примеры склонения

брат → акузативбрата / brata
сто (стол) → акуз.сто / sto
кућа → акузативкућу / kuću
село → акузативсело / selo
Видим брата. / Vidim brata.
Вижу брата.
Читам књигу. / Čitam knjigu.
Читаю книгу.
Идем у школу. / Idem u školu.
Иду в школу (направление).
Важно: Одушевленность важна только для мужского рода! Видим брата (одуш.) но: Видим град (неодуш.).

Используется для обращения

В отличие от русского, в сербском звательный падеж активно используется!

Окончания

м.р.-е, -у ж.р. (-а)-о имена на -о, -е= Ном.

Примеры

Милан →Милане! / Milane!
Марко →Марко! / Marko!
Ана →Ано! / Ano!
господин →господине! / gospodine!
Марко, дођи овамо! / Marko, dođi ovamo!
Марко, иди сюда!
Добар дан, господине! / Dobar dan, gospodine!
Добрый день, господин!
Мајко, где си? / Majko, gde si?
Мама, где ты?

Вопросы: ким? / чим? (кем? / чем?)

Инструмент, средство, совместность. Предлоги: с(а), између, над, под, пред.

Окончания

м./ср. ед.-ом, -ем ж.р. ед.-ом м./ср. мн.-има ж.р. мн.-ама

Примеры склонения

нож → инстр.ножем / nožem
мн.ч.ножевима / noževima
рука → инстр.руком / rukom
мн.ч.рукама / rukama
Пишем оловком. / Pišem olovkom.
Пишу карандашом.
Идем са братом. / Idem sa bratom.
Иду с братом.

Вопросы: где? / о коме? / о чему?

Место, тема разговора. Предлоги: у, на, о, по, при.

Окончания

м./ср. ед.-у ж.р. ед.-и м./ср. мн.-има ж.р. мн.-ама

Примеры склонения

град → локативграду / gradu
мн.ч.градовима / gradovima
школа → локатившколи / školi
мн.ч.школама / školama
Живим у Београду. / Živim u Beogradu.
Живу в Белграде.
Причамо о филму. / Pričamo o filmu.
Говорим о фильме.
Књига је на столу. / Knjiga je na stolu.
Книга на столе.
Важно: «У» и «на» с локативом = где? (у школи). С акузативом = куда? (у школу).
