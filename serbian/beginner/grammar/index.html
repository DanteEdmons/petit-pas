Грамматика — Начальный

Выберите тему, чтобы изучить грамматическое правило с примерами.

Глагол бити (быть) — один из самых важных в сербском. Используется для описания состояния, профессии, национальности.

Утвердительная форма

ја (я)сам / sam
ми (мы)смо / smo
ти (ты)си / si
ви (вы)сте / ste
он/она/оноје / je
они/оне/онасу / su

Отрицательная форма

јанисам / nisam
минисмо / nismo
тиниси / nisi
винисте / niste
он/она/ононије / nije
они/оне/онанису / nisu
Ја сам студент. / Ja sam student.
Я студент.
Она је лекар. / Ona je lekar.
Она врач.
Ми смо пријатељи. / Mi smo prijatelji.
Мы друзья.
Ја нисам уморан. / Ja nisam umoran.
Я не уставший.

Глагол имати (иметь) — обозначает владение, наличие чего-либо.

Утвердительная форма

јаимам / imam
миимамо / imamo
тиимаш / imaš
виимате / imate
он/она/оноима / ima
они/оне/онаимају / imaju

Отрицательная форма

јанемам / nemam
минемамо / nemamo
тинемаш / nemaš
винемате / nemate
он/она/ононема / nema
они/оне/онанемају / nemaju
Ја имам књигу. / Ja imam knjigu.
У меня есть книга.
Она има аутомобил. / Ona ima automobil.
У неё есть автомобиль.
Они немају новац. / Oni nemaju novac.
У них нет денег.
Запомните: В отрицательной форме «не» сливается с глаголом: не + имам = немам.

В сербском вопросы строятся с помощью вопросительных слов или частицы ли.

Вопросительные слова

кто?ко / ko
что?шта / šta
где?где / gde
когда?када / kada
почему?зашто / zašto
как?како / kako
который?који / koji
сколько?колико / koliko
Ко си ти? / Ko si ti?
Кто ты?
Шта радиш? / Šta radiš?
Что делаешь?
Где живиш? / Gde živiš?
Где живёшь?
Како се зовеш? / Kako se zoveš?
Как тебя зовут?

Вопросы с частицей «ли»

Частица ли ставится после глагола для общих вопросов (да/нет).

Да ли говориш српски? / Da li govoriš srpski?
Говоришь ли ты по-сербски?
Јеси ли студент? / Jesi li student?
Ты студент?
Запомните: «Да» в начале вопроса — не означает «yes», это часть вопросительной конструкции.

Глаголы спрягаются в зависимости от окончания инфинитива. Основные типы: -ити и -ати.

говорити / govoriti — говорить (тип -ити)

јаговорим / govorim
миговоримо / govorimo
тиговориш / govoriš
виговорите / govorite
он/она/оноговори / govori
они/оне/онаговоре / govore

волети / voleti — любить (тип -ити)

јаволим / volim
миволимо / volimo
тиволиш / voliš
виволите / volite
он/она/оноволи / voli
они/оне/онаволе / vole

радити / raditi — работать (тип -ити)

јарадим / radim
мирадимо / radimo
тирадиш / radiš
вирадите / radite
он/она/оноради / radi
они/оне/онараде / rade

читати / čitati — читать (тип -ати)

јачитам / čitam
мичитамо / čitamo
тичиташ / čitaš
вичитате / čitate
он/она/оночита / čita
они/оне/оначитају / čitaju
Паттерн окончаний: Глаголы на -ити: -им, -иш, -и, -имо, -ите, -е. Глаголы на -ати: -ам, -аш, -а, -амо, -ате, -ају. Обратите внимание на 3-е лицо мн.ч.!
Ја говорим српски. / Ja govorim srpski.
Я говорю по-сербски.
Она воли кафу. / Ona voli kafu.
Она любит кофе.
Они читају књиге. / Oni čitaju knjige.
Они читают книги.

Для указания направления движения используются предлоги и глаголы движения.

Предлоги направления

ПредлогЗначениеПример
у / uв (внутрь)у школу — в школу
из / izиз (наружу)из куће — из дома
од / odотод куће — от дома
до / doдодо града — до города
ка / kaк, в сторонука центру — к центру
на / naна (поверхность)на посао — на работу
са / saс (поверхности)са посла — с работы

Глаголы движения

идтиићи / ići
приходитьдолазити / dolaziti
уходитьодлазити / odlaziti
путешествоватьпутовати / putovati
Идем у школу. / Idem u školu.
Иду в школу.
Долазим из Београда. / Dolazim iz Beograda.
Приезжаю из Белграда.
Идем на посао. / Idem na posao.
Иду на работу.
Долазим са факултета. / Dolazim sa fakulteta.
Иду с университета.
Запомните: «У» — для закрытых пространств (у школу, у кућу). «На» — для открытых мест или событий (на посао, на састанак).

Књига је на столу. / Knjiga je na stolu.

Кошка в доме.

Лампа је изнад стола. / Lampa je iznad stola.

Кафе је између школе и парка. / Kafe je između škole i parka.
Кафе между школой и парком.
Важно: Предлоги «на» и «у» с локативом отвечают на вопрос «где?», а с акузативом — «куда?».
