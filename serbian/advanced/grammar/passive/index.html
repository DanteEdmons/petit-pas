Пасив (Страдательный залог)

Страдательный залог (Pasiv / Пасив) выражает действие, направленное на подлежащее. Образуется с помощью глагола бити (biti) и страдательного причастия.

бити (biti) в нужном времени + страдательное причастие

Суффиксы страдательного причастия

тип 1 -н (-an) тип 2 -т (-t) тип 3 -ен (-en)
писати → писан / pisati → pisan писать → написанный
пећи → печен / peći → pečen печь → испечённый
видети → виђен / videti → viđen видеть → увиденный

Согласование по роду и числу

м.р. -н / -т / -ен ж.р. -на / -та / -ена ср.р. -но / -то / -ено мн.ч. -ни / -ти / -ени
Књига је написана. / Knjiga je napisana. Книга написана. (ж.р.)
Писмо је написано. / Pismo je napisano. Письмо написано. (ср.р.)
Књиге су написане. / Knjige su napisane. Книги написаны. (мн.ч.)
јесам/си/је/смо/сте/су + причастие
Кућа је грађена сваке године. / Kuća je građena svake godine. Дом строится каждый год.
јесам/си/је/… + био/била/било + причастие
Кућа је била грађена прошле године. / Kuća je bila građena prošle godine. Дом был построен в прошлом году.
Писмо је било написано јуче. / Pismo je bilo napisano juče. Письмо было написано вчера.
ће + бити + причастие
Кућа ће бити грађена следеће године. / Kuća će biti građena sledeće godine. Дом будет построен в следующем году.
Задатак ће бити завршен сутра. / Zadatak će biti završen sutra. Задание будет завершено завтра.
беше/бејаше + причастие
Кућа беше грађена када сам стигао. / Kuća beše građena kada sam stigao. Дом строился, когда я приехал.

Неизвестен или неважен деятель

Прозор је разбијен. / Prozor je razbijen. Окно разбито. (неважно, кем)

Акцент на действии, а не на деятеле

Нови закон је усвојен јуче. / Novi zakon je usvojen juče. Новый закон был принят вчера.

Указание на деятеля: «од стране» / «од»

Для указания на деятеля используется предлог од (od) или од стране (od strane).
Роман је написан од стране позnatog писца. / Roman je napisan od strane poznatog pisca. Роман написан известным писателем.
Кућа је саграђена од мог оца. / Kuća je sagrađena od mog oca. Дом построен моим отцом.
Подлежащее (актив) → «од» + дополнение (пасив)
Дополнение (актив) → Подлежащее (пасив)

Пример 1

Актив: Марија пише писмо. / Marija piše pismo. Мария пишет письмо.
Пасив: Писмо је написано од Марије. / Pismo je napisano od Marije. Письмо написано Марией.

Пример 2

Актив: Ученици су решили задатке. / Učenici su rešili zadatke. Ученики решили задачи.
Пасив: Задаци су решени од ученика. / Zadaci su rešeni od učenika. Задачи решены учениками.
