Пассив (Страдательный залог)

Страдательный залог (Pasiv / Пасив) используется для выражения действия, направленного на подлежащее. В сербском языке пассив образуется с помощью глагола "бити" (biti) и страдательного причастия.

Формула: бити (biti) в нужном времени + страдательное причастие

Страдательное причастие

Образуется от основы инфинитива + суффикс -н/-т/-ен + родовое/числовое окончание:

  • -н: писати → писан (написан) / pisati → pisan
  • -т: пећи → печен (испечен) / peći → pečen
  • -ен: видети → виђен (увиђен) / videti → viđen

Согласование по роду и числу

Мужской род:

Књига је написана. / Knjiga je napisana.

(Книга написана.)

Женский род:

Писмо је написано. / Pismo je napisano.

(Письмо написано.)

Множественное число:

Књиге су написане. / Knjige su napisane.

(Книги написаны.)

Настоящее время (Презент)

Кућа је грађена сваке године. / Kuća je građena svake godine.

(Дом строится каждый год.)

Прошедшее время (Перфект)

Кућа је била грађена прошле године. / Kuća je bila građena prošle godine.

(Дом был построен в прошлом году.)

Писмо је било написано јуче. / Pismo je bilo napisano juče.

(Письмо было написано вчера.)

Будущее время (Футур)

Кућа ће бити грађена следеће године. / Kuća će biti građena sledeće godine.

(Дом будет построен в следующем году.)

Задатак ће бити завршен сутра. / Zadatak će biti završen sutra.

(Задание будет завершено завтра.)

Плюсквамперфект

Кућа беше грађена када сам стигао. / Kuća beše građena kada sam stigao.

(Дом строился, когда я приехал.)

Когда используется пассив

  • Когда неизвестен или неважен деятель:

Прозор је разбијен. / Prozor je razbijen.

(Окно разбито.) [Неважно, кем]

  • Когда акцент на действии, а не на деятеле:

Нови закон је усвојен јуче. / Novi zakon je usvojen juče.

(Новый закон был принят вчера.)

  • Для указания на деятеля используется предлог "од" (от):

Роман је написан од странеpoznatog писца. / Roman je napisan od strane poznatog pisca.

(Роман написан известным писателем.)

Кућа је саграђена од мог оца. / Kuća je sagrađena od mog oca.

(Дом построен моим отцом.)

Схема преобразования

Подлежащее (активный) → Дополнение (пассивный) с предлогом "од"

Дополнение (активный) → Подлежащее (пассивный)

Активный залог:

Марија пише писмо. / Marija piše pismo.

(Мария пишет письмо.)

Страдательный залог:

Писмо је написано од Марије. / Pismo je napisano od Marije.

(Письмо написано Марией.)

Активный залог:

Ученици су решили задатке. / Učenici su rešili zadatke.

(Ученики решили задачи.)

Страдательный залог:

Задаци су решени од ученика. / Zadaci su rešeni od učenika.

(Задачи решены учениками.)

Пример 1:

Аутомобил је купљен прошле недеље. / Automobil je kupljen prošle nedelje.

(Автомобиль был куплен на прошлой неделе.)

Пример 2:

Вечера је спремљена од мајке. / Večera je spremljena od majke.

(Ужин приготовлен мамой.)

Пример 3:

Сва врата су затворена. / Sva vrata su zatvorena.

(Все двери закрыты.)

Пример 4:

Књиге ће бити враћене до петка. / Knjige će biti vraćene do petka.

(Книги будут возвращены до пятницы.)

Пример 5:

Филм је снимљен у Београду. / Film je snimljen u Beogradu.

(Фильм был снят в Белграде.)

Пример 6:

Резултати су објављени јутрос. / Rezultati su objavljeni jutros.

(Результаты были объявлены этим утром.)
